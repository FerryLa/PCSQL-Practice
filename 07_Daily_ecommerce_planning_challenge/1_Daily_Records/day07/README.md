# Day 07 – 운동 루틴 PT 영상 구독

날짜: 2026-02-02

## 1. 한 줄 요약

헬스장에 가도 무엇을 해야 할지 몰라 시간을 낭비하는 사람을 위해, 따라만 하면 되는 맞춤 루틴을 제공하는 구독 서비스

## 2. 이 문제는 언제 발생하는가

* 초보자가 기구 사용법이 헷갈려서 운동 흐름이 끊길 때
* 헬스장에 도착했지만 오늘 어떤 운동을 해야 할지 모를 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 헬스장 트레이너 1:1 PT

**불편 포인트:**
* 영상이 너무 많아 '선택'에서 이미 지침
* 내 체형/목표에 맞는지 확신이 없음

## 4. 핵심 가설 (1개)

만약 사용자 목표와 운동 가능 요일만 입력하면 자동으로 4주 루틴을 완성해주고, 오늘 할 운동만 딱 보여준다면, 사용자는 최소 2주는 운동할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 오운완 )

노출 순서:
1. 오늘 할 운동 카드 리스트
2. 각 운동 클릭 시 30초 핵심 동작 영상 자동 재생
3. 세트 완료 버튼 -> 자동 다음 운동 이동

## 6. 가장 위험한 지점

Retention 위험, 영상이 아무리 좋아도 사람은 운동을 귀찮아 한다.

## 7. 오늘의 기획자 인사이트
콘텐츠의 질보다 시작까지 걸리는 시간을 줄이는 것이 더 중요하다.
