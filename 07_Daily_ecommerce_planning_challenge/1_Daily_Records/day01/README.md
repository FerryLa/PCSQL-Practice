# Day 01 – 육아용품 렌탈 구독

날짜: 2026-01-26
SCAMPER 프레임워크

## 1. 한 줄 요약

육아용품을 렌탈로 해야하는 이유는?

육아용품을 렌탈로 해야하는 이유는 유아기에는 성장이 빨라 체형 변화도 크고, 흥미와 취미도 쉽게 바뀌기 때문에 상품의 소비 측면에서 소유보다 리셀러 가치가 크다

## 2. 이 문제는 언제 발생하는가

* 0~6개월, 체형 변화가 커서 바운서, 아기침대 등이 금방 필요 없어짐
* 6~12개월, 사용하던 제품이 맞지 않음

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 소비자의 중고 거래(리셀)

**불편 포인트:**
* 육아의 부모인 소비자가 리셀의 번거로움을 겪을 때
* 비용이 생각보다 많이 나가고, 집에 육아용품이 쌓일 때

## 4. 핵심 가설 (1개)

만약 육아용품을 필요 기간만 사용하고 반납할 수 있다면, 사용자는 소유 대신 렌탈을 선택할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 성장단계 맞춤 렌탈 선택 화면)

노출 순서:
1. 아이 개월 수 입력 (대상 명확화)
2. 추천 렌탈 제품 3개 제안 (월 비용 표시)
3. “구매 가격 vs 렌탈 비용 비교” 표시 + 렌탈 신청 버튼

## 6. 가장 위험한 지점

민감한 육아 부모가 렌탈 제품을 구매하고 위생적으로 문제가 생겼을 경우입니다.

## 7. 오늘의 기획자 인사이트

소비의 비효율은 개별 선택의 문제가 아니라, 사용 주기와 소유 구조가 맞지 않을 때 발생한다.
