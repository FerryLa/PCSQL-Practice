# Day 12 – 직장인 점심 도시락 구독

날짜: 2026-02-07

## 1. 한 줄 요약

매일 점심 메뉴를 고민하고 결제하는 번거로움을 없애고, 정해진 시간에 사무실로 도착하는 정기 점심 도시락 서비스

## 2. 이 문제는 언제 발생하는가

* 점심시간 직전, 무엇을 먹을지 결정하지 못했을 때
* 배달앱을 열고 메뉴를 고르다가 10~15분을 허비할 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 배달앱 주문, 회사 근처 식당 방문

**불편 포인트:**
* 메뉴 선택에 시간 소모
* 배달 지연 리스크, 점심시간 웨이팅

## 4. 핵심 가설 (1개)

만약 직장인의 점심을 정해진 시간에 자동으로 배송해주고 메뉴 선택을 최소화한다면, 사용자는 편의를 위해 월 단위로 구독할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 3초 구독 시작 화면)

노출 순서:
1. "이번 주 점심, 고민하지 마세요" 한 줄 메시지
2. 가격 옵션 선택 (주 3회 / 주 5회)
3. 회사 주소 입력 후 '이번 주 시작하기' 버튼

## 6. 가장 위험한 지점

직장인은 점심을 "미리 고정하고 싶어하지 않을 가능성"이 크다.
점신은 즉흥성과 사회성이 강한 소비다.

## 7. 오늘의 기획자 인사이트

반복되는 행동이 있다고 해서 반드시 구독이 성립하는 것은 아니다.
