# Day 14 – 중고차 실시간 경매 플랫폼

날짜: 2026-02-09

## 1. 한 줄 요약

사용자는 중고차를 직접 경매에 참여해 실시간으로 최적 가격에 사고팔 수 있다.

## 2. 이 문제는 언제 발생하는가

* 중고차를 사고파는 과정에서 가격 비교와 거래 조건이 불투명할 때
* 딜러를 거치면 시간이 오래 걸리고, 개인 간 거래는 사기 위험이 있을 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 중고차 매매 사이트(딜러 거래)

**불편 포인트:**
* 가격 투명성이 낮아 실제 시세보다 높거나 낮게 거래될 수 있음
* 실시간 경쟁이 없어 판매/구매 속도가 느림

## 4. 핵심 가설 (1개)

만약 사용자가 실시간으로 차량 가격을 보고 즉시 참여할 수 있는 경매 환경이라면, 사용자는 보다 신뢰할 수 있고 효율적으로 차량을 사고팔 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 실시간 경매 참여 화면)

노출 순서:
1. 경매 진행 중인 차량 리스트 + 실시간 가격 업데이트
2. 선택 차량 상세 정보
3. 입찰 버튼 + 내 입찰 현황, 남은 시간 표시

## 6. 가장 위험한 지점

실시간 입찰에 참여한 사용자가 거래를 완료하지 않거나, 가짜 계정으로 가격을 인위적으로 올려 거래 신뢰도가 떨어진 경우

## 7. 오늘의 기획자 인사이트

실시간 경쟁과 투명성은 거래 기반 서비스에서 사용자 참여와 신뢰도를 동시에 높이는 핵심 요소다.
