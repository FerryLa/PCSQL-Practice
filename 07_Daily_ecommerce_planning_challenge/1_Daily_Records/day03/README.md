# Day 03 – 반려동물 사진 앨범 자동 생성 앱

날짜: 2026-01-28

## 1. 한 줄 요약

반려동물 사진을 자동으로 정리와 선별해 스토리형 앨범으로 만들어주는 앱

## 2. 이 문제는 언제 발생하는가

* 반려동물 사진이 넘쳐나서 갤러리에 용량이 초과될 때


## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 스마트폰 기본 앨범에서 직접 선택 후 정리

**불편 포인트:**
* 사진 선별 피로도: 비슷한 사진이 많아 고르는 데 시간과 에너지가 많이 듬


## 4. 핵심 가설 (1개)

만약 앱이 반려동물 사진을 자동으로 선별하고 시간 흐름에 맞춰 스토리 형으로 구성해준다면 사용자의 편의성을 높일 수 있다.

## 5. 최소 화면 1개 설명

(화면 이름: 퍼피 포피)

노출 순서:
1. 이번 달 자동 생성된 앨범 커버
2. 날짜 흐름에 따른 주요 카드
3. 포토북 제작하기, 공유하기

## 6. 가장 위험한 지점

사용자가 자동 선별 결과에 만족하지 못해 신뢰를 잃을 가능성

## 7. 오늘의 기획자 인사이트

자동화 서비스의 경쟁력은 end to end, 사용자가 수정할 필요가 없는 수준까지 가능한 완성도에 달려 있다.
