# Day 05 – 독거노인 말벗 전화 구독2

날짜: 2026-01-30

## 1. 한 줄 요약

나는 오늘도 누군가에게 잊히지 않았다는 느낌을 확인하고 싶다.

## 2. 이 문제는 언제 발생하는가

* 하루 종일 아무와도 말을 하지 않고 저녁이 될 때
* 자녀, 가족에게 연락하고 싶지만 '부담 줄까 봐' 망설일 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 복지관 프로그램 참여

**불편 포인트:**
* 가족에게 자주 연락하기엔 미안함과 눈치
* TV는 '듣는 것'이지 '대화'가 아님

## 4. 핵심 가설 (1개)

만약 같은 상담원이 정기적으로 전화를 걸어주고 이전 대화를 기억해준다면,
사용자는 이 서비스를 단순 안부가 아니라 '관계'로 인식하고 장기 구독할 것이다.

핵심은 빈도보다 지속적 관계성

## 5. 최소 화면 1개 설명

(화면 이름: 오늘의 전화 현황)

노출 순서: 오늘 통화 예정 시간 표시
1. 오늘 오후 4시, 김영희 상담사님과 통화 에정입니다.
2. 지난 통화: 손주 사진 이야기 나눔
3. 통화 주제 미리 선택 버튼: 건강 이야기, 뉴스 이야기, 그냥 수다

"나는 누군가에게 기억되고 있다'는 확신 제공

## 6. 가장 위험한 지점

운영 리스크 1순위: 상담사 품질과 지속성
- 상당사가 자주 바뀌면 관계 자산이 무너짐
- 감정노동으로 번아웃 가능성 높음
- 형식적인 통화과 되면 신뢰 붕괴

## 7. 오늘의 기획자 인사이트

외로움은 시간의 문제가 아니라 관계의 밀도 문제다.
