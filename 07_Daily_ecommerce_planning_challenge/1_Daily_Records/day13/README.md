# Day 13 – 헬스 식단 도시락 구독

날짜: 2026-02-08

## 1. 한 줄 요약

운동은 하지만 식단을 직접 관리하기 어려운 사람들을 위해, 목표에 맞는 헬스 식단을 정기 배송해주는 서비스

## 2. 이 문제는 언제 발생하는가

* 운동을 시작했지만 식단을 어떻게 구성해야 할지 모를 때
* 다이어트/벌크업 중인데 외식이나 배달 유혹을 이기기 어려울 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 직접 장을 보고 식단을 만들어 먹는다.

**불편 포인트:**
* 메뉴가 단조롭고 금방 질린다.
* 매번 영양소 계산이 번거롭다.

## 4. 핵심 가설 (1개)

만약 운동 목표에 맞춘 맞춤형 식단을 자동으로 추천하고 정기 배송해준다면, 사용자는 식단 스트레스 없이 운동을 더 오래 지속할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 목표 기반 식단 선택 화면)

노출 순서:
1. 목표 선택 (체지방 감량 / 근육 증가 / 유지)
2. 하루 칼로리, 단백질 자동 계산 결과 표시
3. 추천 식단 패키지 + 주간 구독 가격 + 정기 구독 시작

## 6. 가장 위험한 지점

식단의 맛이 없거나 메뉴가 반복되면 바로 이탈할 가능성이 높다.
즉, 문제는 기능이 아니라 지속 가능한 만족도다.

## 7. 오늘의 기획자 인사이트

사람은 '식단을 잘하고 싶어하는 것'이 아니라 '결정을 덜 하고 싶어한다.'
자동화가 가치다
