# Day 10 – 집 근처 공유 주방 대여

날짜: 2026-02-05

## 1. 한 줄 요약

집에서는 해결할 수 없는 '요리 상황'을 빠르게 해결하기 위해 근처의 전문 주방을 빌려 쓰는 서비스

## 2. 이 문제는 언제 발생하는가

* 집에서 요리하기에 냄새와 공간 제약이 부담스러울 때
* 대량으로 조리가 필요한데 장비가 부족할 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 집에서 무리해서 조리

**불편 포인트:**
* 가정용 주방은 화구와 조리 공간이 부족함
* 냄새와 뒷정리 부담이 큼

## 4. 핵심 가설 (1개)

만약 집 근처에서 시간 단위로 간편하게 예약 가능한 공유 주방이 있다면, 사용자는 특정 요리 상황을 해결하기 위해 기꺼이 비용을 지불할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 우리 동네 주방 예약하기 )

노출 순서:
1. 현재 위치 기반 근처 공유 주방 리스트
2. 시간 선택 + 예상 가격 표시
3. 장비 정보 및 즉시 예약 버튼

## 6. 가장 위험한 지점

주방 위생 관리 및 설비 파손 리스크가 높아 운영 비용이 예상보다 크게 발생할 가능성이 있음

## 7. 오늘의 기획자 인사이트

집에서 해결되지 않는 순간을 해결하기 위해 공간을 구매한다.
