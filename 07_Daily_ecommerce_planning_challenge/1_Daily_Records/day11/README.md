# Day 11 – 중고 아이폰 실시간 시세 조회

날짜: 2026-02-06

## 1. 한 줄 요약

리셀이 활발한 중고 아이폰의 가격을 실시간으로 알아보는 것
-- 사용자가 중고 아이폰을 판매하거나 구매할 때 실시간 시세를 즉시 확인할 수 있는 서비스

## 2. 이 문제는 언제 발생하는가

* 턱없이 비싼 아이폰을 살 돈이 없지만 고민할 때
* 기존에 쓰던 아이폰에서 새로운 아이폰으로 바꾸고 싶을 때
-- 중고 아이폰을 팔거나 살 때, 적정 가격을 알기 어려울 때
-- 중고 시세가 시시각각 변해서 최신 정보가 필요할 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 당근마켓 등에서 수동으로 유사 모델 시세 검색

**불편 포인트:**
* 실시간 시세 확인이 어렵고, 검색 과정이 번거롭다.
* 모델, 용량, 상태에 따른 가격 편차를 직접 계산해야 한다.

## 4. 핵심 가설 (1개)

만약 모델, 용량, 상태를 입력하면 실시간으로 판매/구매 시세를 보여주는 기능이 있다면, 사용자는 빠르게 합리적인 가격으로 중고 거래를 결정할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 중고 시세 조회)

노출 순서:
1. 아이폰 모델 선택 (예: iPhone 13, 14 등)
2. 저장 용량 선택 (예: 126GB, 256GB)
3. 실시간 시세 확인: 최근 거래가, 평균 시세 등

## 6. 가장 위험한 지점

실시간 시세 데이터 정확도 문제로 업데이트가 늦거다 오류가 발생하면 사용자가 잘못된 가격으로 거래할 가능성이 있다.

## 7. 오늘의 기획자 인사이트

실시간 데이터와 사용자의 의사결정 속도는 서비스 만족도에 직결된다.
