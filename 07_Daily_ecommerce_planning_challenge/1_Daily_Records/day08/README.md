# Day 08 – 육아 일기 자동 기록 앱

날짜: 2026-02-03
JTBD 관점 기획

## 1. 한 줄 요약

나는 아이의 성장 순간을 놓치지 않고 남기고 싶지만, 매번 기록할 여유는 없다.

## 2. 이 문제는 언제 발생하는가

* 하루가 끝났는데 "오늘 뭐 했지?" 기억이 흐릿할 때
* 사진은 많은데 정리되지 않아 방치될 때
* 나중에 아이에게 보여줄 기록이 없을 것 같아 불안할 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 인스타 비공개 계정
* 카카오톡 나와의 채팅방

**불편 포인트:**
* 매번 "써야 한다"는 의식적인 노력 필요
* 사진가 텍스트가 분리되어 있음

## 4. 핵심 가설 (1개)

만약 사진, 음성, 메신저 대화를 기반으로 자동으로 하루를 요약해준다면, 사용자는 의지 없이도 육아 기록을 지속할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 오늘 자동 기록)

노출 순서:
1. 오늘 찍은 사진 자동 정렬
2. AI가 생성한 하루 요약 문장
3. [수정하기] / [그대로 저장] 버튼

## 6. 가장 위험한 지점

부모는 "자동 생성된 감정 표현"이 진짜 내 감정처럼 느껴지지 않으면 거부감을 가질 수 있다.

## 7. 오늘의 기획자 인사이트

사람들은 기록을 원하지만, 기록하는 행동은 원하지 않는다.
