# Day 04 – 독거노인 말벗 전화 구독

날짜: 2026-01-29

## 1. 한 줄 요약

정기 구독 기반으로 전문 상담 교육을 받은 말벗 매니저가 독거노인에게 주기적으로 전화를 제공해주는 정서 케어 서비스

## 2. 이 문제는 언제 발생하는가

* 자녀가 타 지역에 거주하여 일상적 대화가 단절된 상황
* 거동이 불편해 외부 활동이 어려운 경우

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 종교 단체 방문 봉사

**불편 포인트:**
* 운영 시간 제한
* 가족에게 정서적 부담 전가

## 4. 핵심 가설 (1개)

만약 정기적이고 관계 기반의 말벗 통화를 제공한다면, 보호자는 비용을 지불해서라도 부모의 정서 안정 서비스를 구독할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 부모 안심 구독 신청)

노출 순서: 부모님과 하루 몇 분 통화하시나요?
1. 서비스 설명(주 3회, 10분 통화)
2. 가격 플랜 선택 (월 구독)
3. 첫 통화 예약

## 6. 가장 위험한 지점

대화 역량이 낮으면 고객 이탈 발생 / 운영 난이도가 리스크다.

## 7. 오늘의 기획자 인사이트

감성 문제일수록 수익 구조의 운영 구조를 먼저 설계해야 지속 가능합니다.
