# Day 09 – 여행지 짐 보관 중개 플랫폼

날짜: 2026-02-04

## 1. 한 줄 요약

체크인 전, 체크아웃 후 애매한 시간, 여행자의 짐을 근처 상점에 잠시 맡겨주는 위탁 중개 서비스

## 2. 이 문제는 언제 발생하는가

* 숙소 체크인 전 도착했을 떄 (오전 도착 항공편)
* 환승 도시에서 몇 시간 공백이 생겼을 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 숙소에 부탁

**불편 포인트:**
* 숙소가 거절하거나 시간 제약이 있음
* 보관 가능 시간, 가격 정보가 실시간이 아님

## 4. 핵심 가설 (1개)

만약 여행 동선 근처 상점들이 실시간으로 짐 보관 가능 여부를 공개하고 간편 예약 가능하다면, 사용자는 짐을 들고 다니는 대신 비용을 지불하고 맡길 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 지도 기반 즉시 보관 찾기 화면)

노출 순서:
1. 현재 위치 기반 지도 + 주변 보관 가능 상점 핀 표시
2. 가격 / 운영시간 / 최대 보관 가능 크기 표시
3. 결제 후 QR 발급

## 6. 가장 위험한 지점

상점이 짐 분실/훼손했을 때 책임 문제

## 7. 오늘의 기획자 인사이트

공간이 부족한 게 아니라, 신뢰와 연결이 부족한 경우가 더 많다.
