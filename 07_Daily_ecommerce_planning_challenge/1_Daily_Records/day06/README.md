# Day 06 – 개인 트레이너 화상 PT 구독

날짜: 2026-02-01
Learn Canvas 스타일

## 1. 한 줄 요약

헬스장에 가지 않아도 전문 트레이너와 꾸준히 운동할 수 있다.

## 2. 이 문제는 언제 발생하는가

* 여유가 없어 헬스장에 가지 못할 때
* 운동법을 몰라 혼자 하면 동기부여가 떨어질 때

## 3. 기존 해결 방식과 불편함

**기존 대안:**
* 헬스장 등록 후 개인 트레이너 수업
* 유트브, 앱을 통한 홈트레이닝 영상 시청

**불편 포인트:**
* 헬스장 PT는 비용이 많이 들고, 시간과 장소의 제약이 크다.
* 영상 기반 PT는 실시간 피드백이 없어 자세 교정을 하기 어렵다.

## 4. 핵심 가설 (1개)

만약 전문 트레이너와 화상 PT를 할 수 있다면, 사용자는 꾸준히 운동하며 만족할 것이다.

## 5. 최소 화면 1개 설명

(화면 이름: 운동 세션 시작 화면)

노출 순서: 운동 세션 시작 화면
1. 예약된 트레이너와 세션 화면
2. '세션 시작' - '화상 연결' 버튼 -> 화상 연결
3. 운동 기록 자동 저장 및 피드백 노트 확인

## 6. 가장 위험한 지점

화상 PT 경험이 실제 오프라인 PT 대비 충분히 만족스럽지 않을 경우 -> 화면 상 기술적인 문제나 PT트레이너의 화상 서비스 방법이 효율적이지 못할 때

## 7. 오늘의 기획자 인사이트

서비스의 지속 가능성을 좌우하는 건 ‘참여율’이 아니라 ‘핵심 경험이 진짜로 가치 있는가’이다.
