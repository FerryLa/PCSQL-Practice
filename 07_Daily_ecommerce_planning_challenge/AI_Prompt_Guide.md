
# 100일 이커머스 기획 챌린지

## 일일 기획용 AI 프롬프트 가이드

전제
이미 7가지 항목으로 일일 기획을 완료한 상태에서 사용하는 보조 프롬프트입니다.

목적

* 사고 밀도 강화
* 리스크 조기 발견
* 사업 관점 확장
* 포트폴리오 문장 정제

---

## 1. AI 반박 프롬프트 (기본 사용)

```
아래는 오늘 작성한 이커머스 기획안이다.

[여기에 7가지 항목 전체 붙여넣기]

1. 이 아이디어가 실패할 가능성이 높은 이유 5가지를 냉정하게 말해줘.
2. 가장 위험한 가설 하나를 지정하고 왜 치명적인지 설명해줘.
3. 이 아이디어를 최소 비용으로 빠르게 검증하려면 어떤 실험을 하면 좋을지 제안해줘.
```

---

## 2. AI 확장 프롬프트 (사업 관점 강화)

```
아래 기획안을 기반으로 사업 관점에서 확장해줘.

[기획안 붙여넣기]

1. 타겟 고객을 3개 세그먼트로 구체화해줘.
2. 수익 모델을 3가지 제안해줘.
3. 기존 경쟁 서비스 대비 차별화 포인트 5개를 정리해줘.
```

---

## 3. AI 압축 프롬프트 (핵심 정리 훈련)

```
아래 기획안을 투자자 피치용으로 3문장으로 압축해줘.
그리고 한 문장 슬로건도 만들어줘.

[기획안 붙여넣기]
```

## 4. JTBD 검증 프롬프트

```
이 아이디어가 해결하려는 고객의 Job을
기능적 / 감정적 / 사회적 측면에서 각각 정리해줘.

그리고 이 Job이 충분히 강력한 문제인지 평가해줘.

[기획안 붙여넣기]
```

## 5. AARRR 퍼널 분석 프롬프트

```
이 아이디어가 AARRR 퍼널 중 어느 단계에 가장 크게 기여하는지 분석해줘.
그리고 퍼널 전체 관점에서 보완해야 할 부분을 제안해줘.

[기획안 붙여넣기]
```
