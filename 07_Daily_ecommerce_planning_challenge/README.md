# 🏃 100일 이커머스 기획 챌린지

## 📌 목적
매일 하나의 이커머스 아이디어를 기획하여 실무 감각을 키우고 포트폴리오를 구축합니다.

---

## 📂 폴더 구조

```
07_Daily_ecommerce_planning_challenge/
├── 1_Daily_Records/          # 일별 기획 파일 (day01 ~ day100)
├── 2_30day_Summary/          # 30일 회고 (30일, 60일, 90일) #  2_Weekly_Summary/  # 주간 회고 (week01 ~ week14)
├── 3_Idea_Pool/               # 아이디어 풀 (ideas.txt, used_ideas.txt)
├── 4_Portfolio_Drafts/        # 포트폴리오 초안 (베스트 기획 모음)
├── 5_Final_Portfolio/         # 최종 포트폴리오 (Top 10 정리)
└── run_today.sh               # 오늘의 챌린지 자동 실행 스크립트
```

---

## 🚀 사용법

### 1. 오늘의 챌린지 시작
```bash
bash run_today.sh
```

이 스크립트는:
- `3_Idea_Pool/ideas.txt`에서 **사용하지 않은** 랜덤 아이디어 하나를 선택
- `1_Daily_Records/dayXX/README.md` 생성
- 사용한 아이디어를 `used_ideas.txt`에 기록
- 남은 아이디어 개수 표시

### 2. 기획 작성
생성된 `dayXX/README.md` 파일을 열어 7가지 항목을 작성합니다:
1. 한 줄 요약
2. 문제 발생 시점
3. 기존 해결 방식과 불편함
4. 핵심 가설
5. 최소 화면 1개 설명
6. 가장 위험한 지점
7. 오늘의 기획자 인사이트

### 3. 30일 회고 (선택)
<!-- 매 주마다 `2_Weekly_Summary/XXday.md`에 학습 내용과 변화를 기록하고 정리합니다. -->
 매 30일마다 `2_30day_Summary`에 학습 내용과 변화를 기록하고 정리합니다.

### 4. 포트폴리오 준비 (80일차 이후)
- 베스트 기획 10개를 `4_Portfolio_Drafts/`에서 다듬기
- 최종본을 `5_Final_Portfolio/`에 정리

---

## 🎯 챌린지 규칙

1. **하루도 빠짐없이** – 100일 연속 기록
2. **15분 안에 완성** – 빠른 사고력 훈련
3. **실제 문제 중심** – 가상이 아닌 진짜 불편함 해결
4. **중복 없이 진행** – `used_ideas.txt`로 자동 관리

## 📊 진행 상황

- **총 아이디어**: 100개
- **오늘 Day**: 자동 생성됨
- **남은 아이디어**: `run_today.sh` 실행 시 표시
