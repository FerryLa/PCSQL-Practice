# 09_battery-factory 

Claude와 함께 만든 **브라우저 기반 배터리 공장 운영 게임**입니다.  
제한 시간 안에 자원을 관리하고, 생산 라인을 최적화해 목표 점수를 달성하는 것을 목표로 합니다.

- Claude 공유 링크: https://claude.ai/share/dce5a9bc-4a62-47ee-9638-c818709c31a8

---

## 게임 개요

`battery-factory`는 공장 운영 시뮬레이션의 핵심 재미를 가볍게 체험할 수 있도록 만든 미니 게임입니다.

- 자원(원재료/에너지)을 효율적으로 사용
- 배터리 생산량을 높이고 불량률을 낮추기
- 라운드가 진행될수록 난이도 상승

---

## 플레이 방법

1. 게임을 실행하면 초기 자원과 목표가 제시됩니다.
2. 생산 버튼/설비 제어를 통해 배터리를 생산합니다.
3. 자원 고갈, 설비 과열/다운타임(구현된 경우)을 관리합니다.
4. 제한 시간(또는 턴) 종료 시 점수가 계산됩니다.
5. 목표 점수 달성 여부에 따라 승/패가 결정됩니다.

---

## 핵심 규칙 (예시)

- 배터리 1개 생산 시 자원이 소모됩니다.
- 자원이 부족하면 생산이 중단됩니다.
- 점수는 `생산량`, `효율`, `잔여 자원`, `패널티` 등을 반영합니다.
- 난이도가 올라갈수록 자원 소모량 또는 이벤트 빈도가 증가합니다.

> 실제 게임 로직은 코드 구현 기준으로 달라질 수 있습니다.
